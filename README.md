# rtc_ntp
Init RTC Time from NTP

## How to Use
- Clone / Download this repository to your pc
- Open rtc_ntp.ino with Arduino Sketch
- Set Board to your ESP3266 Device (Wemos, NodeMCU, etc)
- Set Port
- Upload
- Done

## Pin Configuration
Wemos Pin Configuration
D1 - SCL
D2 - SDA
3V3 - VCC
GND - GND
